# iOS Configuration (Future Mobile Companion App)

This directory will contain iOS-specific configuration for the Family Calendar mobile companion app.

## Requirements
- Xcode 14.0 or later
- iOS 14.0+ deployment target
- CocoaPods for dependency management

## Note

iOS implementation is planned for Phase 7 (Mobile Companion App).
The main application targets Android Mini PC devices.
The iOS app will be a companion app for family members to:
- View and edit calendar events on mobile
- Add and complete chores
- Manage shopping lists
- View meal plans
- Receive push notifications

## Future Build Commands

```bash
cd ios
pod install
cd ..
npm run ios
```

This directory is a placeholder for future development.
