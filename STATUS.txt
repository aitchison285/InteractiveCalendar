API KEYS REQUIRED
================================================================================

Before running the application, you need to obtain and configure:

1. Google Calendar API
   - Web Client ID and Android Client ID
   - OAuth 2.0 configuration
   - Enable Google Calendar API in Google Cloud Console

2. Microsoft Graph API (Outlook)
   - Application ID and Client Secret
   - Azure AD app registration
   - Calendars.ReadWrite permission

3. Apple Calendar (CalDAV)
   - Apple ID email
   - App-specific password (generate in Apple ID settings)

4. Recipe APIs (at least one)
   - TheMealDB API key (free tier available)
   - OR Spoonacular API key
   - OR Edamam API key

5. Weather API (Optional)
   - OpenWeatherMap API key

6. Firebase (Optional, for cloud sync)
   - Firebase project configuration
   - Cloud Firestore
   - Firebase Authentication
   - Cloud Messaging for notifications

INSTALLATION INSTRUCTIONS FOR FUTURE COPILOT AGENTS
================================================================================

1. DEPENDENCIES INSTALLATION:
   ```bash
   cd /home/runner/work/InteractiveCalendar/InteractiveCalendar
   npm install
   ```

2. ENVIRONMENT CONFIGURATION:
   - Copy .env.example to .env
   - Add all required API keys to .env file

3. ANDROID SETUP:
   ```bash
   cd android
   ./gradlew clean
   cd ..
   ```

4. RUN DEVELOPMENT BUILD:
   ```bash
   npm start          # Start Metro bundler
   npm run android    # Run on Android device/emulator
   ```

5. BUILD PRODUCTION APK:
   ```bash
   npm run build:android
   # APK location: android/app/build/outputs/apk/release/
   ```
